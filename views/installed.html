Session Alive Installed

Thanks for installing Session Alive extension! Create a rule to keep your Session Alive!

How to use

  1. Sign-in to your website and Copy Dashboard/Welcome URL
  2. Click the button in the Session Alive extension pop-up
  3. Click the button to add a New Rule and paste the copied URL to Background Request Options → Trigger URL
  4. Set the Request Interval as per your needs and click the button to save.
  5. Click the Trigger URL link in the Save Success message. Navigate to Trigger URL to start the Rule.
  6. You can also Refresh the Dashboard/Welcome URL or Sign-in again
  7. You can open the trigger URL in Container Tabs. Right-click Trigger URL link and select Open Link in New Multi-Account Container Tab
  8. Your session is now alive!

Quick Start

  1. Click the Session Alive extension icon in Firefox toolbar or overflow menu
  2. Click the button
  3. Enter Rule Details in the Add New Rule page
    • Rule Name: Quick Start
    • Background Request Options → Trigger URL: https://example.com/
    • Background Request Options → Request Interval (in minutes): 1
  4. Click the button
  5. Click the Trigger URL link in the Save Success message. Navigate to Trigger URL to start the Rule.
